1. Prompt the user to input a text string and store it in the 'text' variable.
2. Function count_letters(text):
   - Initialize a counter 'count' to 0.
   - Loop through each character in 'text':
      * If it's alphabetic, increment 'count' by '1'.
   - Return 'count'.
3. Function count_words(text):
   - Initialize a counter 'count' to '1' to account for the first word.
   - Loop through each character in 'text':
      * If it's a whitespace, increment 'count' by '1'.
   - Return 'count'.
4. Function count_sentences(text):
   - Initialize a counter 'count' to 0.
   - Loop through each character in 'text':
      * If it's a punctuation character (., !, ?), increment 'count' by '1'.
   - Return 'count'.
5. Count the number of letters, words, and sentences in the input text
   - Call count_letters(text) and store the result in 'letters_count'.
   - Call count_words(text) and store the result in 'words_count'.
   - Call count_sentences(text) and store the result in 'sentences_count'.
8. Calculate the Coleman-Liau index (CLI):
   - Calculate the average number of letters per 100 words (L).
   - Calculate the average number of sentences per 100 words (S).
   - Calculate the index using the formula: index = 0.0588 * L - 0.296 * S - 15.8.
9. Determine the reading level based on the CLI:
   - If the index is greater than 16, print "Grade 16+\n".
   - If the index is less than 1, print "Before Grade 1\n".
   - Otherwise, print "Grade X\n" where X is the integer value of the index.
